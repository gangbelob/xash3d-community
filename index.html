Android Playable Mods

Created By: MrArab

Xash3D FWGS is an awesome engine that allows you to play classic GoldSrc Half-Life mods natively on your Android device. Below are some recommended compatible choices:

Blue Shift

Step into the security guard boots of Barney Calhoun as you fight to secure escape routes for Black Mesa scientists.

dir: bshift

Opposing Force

Play as Corporate Soldier Adrian Shephard sent to handle weapon systems and neutralize dangerous alien threat assets.

dir: gearbox

Afraid of Monsters DC

A classic creepy conversion based inside a surreal, nightmarish hospital layout. Extreme intense atmosphere options.

dir: aomdc

USS Darkstar

Command survival operations tracking containment hazards inside a massive research ship lightyears away.

dir: darkstar

Cry of Fear Mobile

A brutal psychological horror campaign conversion. Complete custom weapons, tracking inventory layouts, and dark atmospheric cities.

dir: cof

Counter-Strike 1.6

The legendary classic tactical multi-player framework port running smoothly alongside standard touch configurations.

dir: cstrike

They Hunger

A massive retro zombie singleplayer campaign. Fight through post-apocalyptic towns, secret laboratories, and swamps.

dir: hunger

Paranoia

Play as a Russian alpha tactical operative handling classified anomalies inside an industrial underground base.

dir: paranoia

How to Install & Play Xash3D Mods

  • Step 1: Download Mod Files — Download mod zip files from reliable databases like ModDB.
  • Step 2: Initialize Engine — Install the core application engine Xash3D FWGS app package onto your phone.
  • Step 3: Structure Data Directory — Using your mobile file manager, create an asset folder labeled xash in internal storage. Paste your raw game data folder (valve) inside.
  • Step 4: Extract Mod Assets — Unzip your mod file and drop the customized mod directory (Example: aomdc) directly next to your base valve file structure inside xash/.
  • Step 5: Define Launch Arguments — Boot Xash3D, point it to your data directory, input the specific mod directory name in the custom parameters prompt, and start the engine!
